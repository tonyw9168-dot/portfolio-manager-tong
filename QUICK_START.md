# 快速开始指南

## 项目修改概览

本次修改为"桐的家庭基金"项目实现了三项主要功能改进：

### 1️⃣ 资产分析页面 - 详细明细展示
- **位置**: 左侧菜单 → 资产分析
- **新功能**: 点击任何资产类别卡片即可展开查看详细资产明细
- **显示内容**: 资产名称、当前价值、占比、变化率、买入时间等

### 2️⃣ 性能分析改名为收益分析
- **位置**: 左侧菜单 → 收益分析（原"性能分析"）
- **变更**: 页面名称、导航菜单、移动端菜单都已更新

### 3️⃣ 数据增删改功能
- **现金流管理**: 支持新增、编辑、删除现金流记录
- **配置规划**: 支持新增、编辑、删除配置类别
- **仪表板**: 支持快速新增资产

## 安装和运行

### 前置要求
- Node.js 20+
- pnpm 包管理器

### 安装依赖
```bash
cd portfolio-manager-tong
pnpm install
```

### 开发模式
```bash
pnpm dev
```
访问 http://localhost:3000

### 生产构建
```bash
pnpm build
```

## 主要页面功能

### 📊 资产分析页面
| 功能 | 说明 |
|------|------|
| 按类别视图 | 点击类别卡片展开查看资产明细 |
| 按标的视图 | 按单个资产查看详情 |
| 按时间视图 | 查看资产随时间的变化趋势 |
| 类别对比矩阵 | 表格形式对比各类别数据 |

### 💰 现金流管理
| 功能 | 操作 |
|------|------|
| 新增现金流 | 点击"新增现金流"按钮 |
| 编辑记录 | 表格右侧下拉菜单 → 编辑 |
| 删除记录 | 表格右侧下拉菜单 → 删除 |
| 数据统计 | 自动计算总流入、总流出、净流量 |

### 🎯 配置规划
| 功能 | 操作 |
|------|------|
| 新增类别 | 点击"新增类别"按钮 |
| 编辑比例 | 配置详情列表 → 下拉菜单 → 编辑 |
| 删除类别 | 配置详情列表 → 下拉菜单 → 删除 |
| 查看配置 | 饼图显示配置分布，卡片显示平衡状态 |

## 常见问题

### Q: 如何在移动端查看？
A: 使用浏览器开发者工具的设备模拟功能，或直接在手机上访问。所有页面都已适配移动端。

### Q: 数据会自动保存吗？
A: 是的，所有操作都会通过 API 自动保存到数据库。

### Q: 如何撤销误操作？
A: 删除操作需要确认，可以防止误删。编辑操作可以重新编辑覆盖。

### Q: 支持哪些币种？
A: 现金流管理支持 CNY（人民币）、USD（美元）、HKD（港币）。

### Q: 如何导出数据？
A: 仪表板页面有"导出Excel"按钮，可以导出投资数据。

## 技术栈

- **前端框架**: React 18 + TypeScript
- **UI 组件**: shadcn/ui
- **样式**: Tailwind CSS
- **图表**: Recharts
- **状态管理**: React Query (tRPC)
- **后端**: Node.js + Express
- **数据库**: MySQL/TiDB
- **ORM**: Drizzle

3. **代码分割**: 按需加载页面组件
